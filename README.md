## Tech Stack

- React.js
- Sass
- Css animations
- Media queries for mobile responsive

### Installation

1- Clone this repo  
2- Install the dependencies and devDependencies and start the application.

#### Using NPM

```sh
$ cd beats-music
$ npm install
$ npm run start
```
